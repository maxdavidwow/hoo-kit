# hoo-kit

hoo-kit provides a Task-System which handles shell execution based on events.<br/>

## Get Started
Install:
<br/>
`npm install @wow-kit/hoo-kit`
<br/>
<br/>
Run:
<br/>
*from a node module*
<br/>
```javascript
import runHookit from 'hoo-kit';
runHookit();
```
*from the cli just run*
<br/>
```
hoo-kit
```
